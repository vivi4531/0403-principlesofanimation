Example:

Make the beachball follow the path of the blue ball in the gif-animation.
Don't worry about synchronizing their movement, you can't do that with a gif - but it takes approximately 2.58 seconds to complete.
Start without any animation, and identify the extremes of the path - the start and the end. Use transform: translate( ) with vw as units. Keep all the transform-lines, but comment each one out, as you try to find the next one.
After having found the extremes, find the top and bottom of each 'bounce' along the path.

When you have enough points, make a @keyframes animation, and put all the transforms at various times - use the x-axis on the path as a guide: 0% would be the start, 100% the end, the first bounce around 30-35%.
Use linear timing-function, and see how close (or not) the animation follows the path.

If you want to improve the animation, you'll have to use different timing on x and y-axis - this will be in part 9. Timing!
But you can revisit this exercise, and make the container do the x-translation, and make the ball only do the y-translation.
x would still be linear.
